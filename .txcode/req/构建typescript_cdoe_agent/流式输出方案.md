# 事件 + WebSocket 混合流式方案

## 1. 方案概述

AI Agent 流式输出（Thought/Action/Observation）采用**混合架构**：
- **CLI 模式**：直接事件回调（无网络开销）
- **Web 模式**：WebSocket 推送（支持浏览器）

```
┌─────────────────────────────────────────────────────────────────┐
│                        API 层 (统一入口)                          │
│                                                                 │
│  chatWithAgent(message, callbacks)                             │
│         │                                                       │
│         ├── callbacks.onThought?.(thought)                      │
│         ├── callbacks.onAction?.(action, input)                 │
│         ├── callbacks.onObservation?.(result)                   │
│         └── callbacks.onFinalAnswer?.(answer)                    │
│                                                                 │
└─────────┬───────────────────────────────────────────────────────┘
          │
          ├─── CLI 模式 ────► 直接调用 callbacks，实时打印
          │
          └─── Web 模式 ────► WebSocket 推送，浏览器接收
```

## 2. 事件类型定义

```typescript
// modules/ai/react/react.types.ts

export type AgentEventType =
  | 'thought'      // AI 思考过程
  | 'action'       // AI 调用工具
  | 'observation'  // 工具执行结果
  | 'final_answer' // 最终回答
  | 'error'        // 错误
  | 'done';        // 完成

export interface AgentEvent {
  type: AgentEventType;
  data: any;
  stepIndex?: number;
  timestamp: number;
}

export interface ThoughtEvent {
  type: 'thought';
  data: {
    thought: string;
    stepIndex: number;
  };
}

export interface ActionEvent {
  type: 'action';
  data: {
    toolName: string;
    actionInput: Record<string, any>;
    stepIndex: number;
  };
}

export interface ObservationEvent {
  type: 'observation';
  data: {
    result: any;
    isError?: boolean;
    stepIndex: number;
  };
}

export interface FinalAnswerEvent {
  type: 'final_answer';
  data: {
    answer: string;
  };
}
```

## 4. ReAct Agent 实现

```typescript
// modules/ai/react/react.agent.ts

import { AgentCallbacks, ReActStep, ReActResult } from './react.types';

export class ReActAgent {
  private maxIterations = 10;

  async run(
    userMessage: string,
    context: { projectPath: string; sessionId: string },
    callbacks?: AgentCallbacks
  ): Promise<ReActResult> {
    const steps: ReActStep[] = [];
    const historyMessages: ChatMessage[] = [];
    let stepIndex = 0;

    // ... 初始化 system prompt ...

    while (iteration < this.maxIterations) {
      iteration++;

      // 获取 AI 响应
      const response = await aiProvider.chat(historyMessages);
      const { steps: newSteps, finalAnswer } = reactParser.parse(response.content);

      // 处理新步骤
      for (const newStep of newSteps.slice(steps.length)) {
        stepIndex++;

        if (newStep.final_answer) {
          callbacks?.onFinalAnswer?.(newStep.final_answer);
          callbacks?.onDone?.();
          break;
        }

        if (newStep.thought) {
          callbacks?.onThought?.(newStep.thought, stepIndex);
        }

        if (newStep.action) {
          callbacks?.onAction?.(newStep.action, newStep.actionInput, stepIndex);

          // 执行工具
          const toolResult = await this.executeTool(newStep.action, newStep.actionInput, context);
          newStep.observation = toolResult.success ? toolResult.data : { error: toolResult.error };

          callbacks?.onObservation?.(newStep.observation, !toolResult.success, stepIndex);
        }
      }

      if (reactParser.hasFinalAnswer(steps)) break;
      historyMessages.push({ role: 'assistant', content: aiContent });
    }

    callbacks?.onDone?.();

    return {
      answer: finalAnswer || '',
      steps,
      iterations: iteration,
      success: iteration < this.maxIterations,
    };
  }
}

export const reactAgent = new ReActAgent();
```

## 6. Web 服务（WebSocket）

```typescript
// modules/web/web.service.ts

import { Server } from 'socket.io';
import { createServer } from 'http';
import { AgentCallbacks, AgentEvent } from '../modules/ai/react/react.types';
import { chatWithAgent } from '../api/ai.api';

export class WebService {
  private io: Server;
  private httpServer: ReturnType<typeof createServer>;

  constructor(port = 40000) {
    this.httpServer = createServer();
    this.io = new Server(this.httpServer, {
      cors: { origin: '*' }
    });
    this.setupWebSocket();
    this.httpServer.listen(port);
  }

  private setupWebSocket() {
    this.io.on('connection', (socket) => {
      console.log('Client connected:', socket.id);

      socket.on('chat', async (data: { message: string; context?: any }) => {
        const callbacks: AgentCallbacks = {
          onThought: (thought, stepIndex) => {
            socket.emit('agent_event', {
              type: 'thought',
              data: { thought, stepIndex },
              timestamp: Date.now()
            } as AgentEvent);
          },
          onAction: (toolName, actionInput, stepIndex) => {
            socket.emit('agent_event', {
              type: 'action',
              data: { toolName, actionInput, stepIndex },
              timestamp: Date.now()
            } as AgentEvent);
          },
          onObservation: (result, isError, stepIndex) => {
            socket.emit('agent_event', {
              type: 'observation',
              data: { result, isError, stepIndex },
              timestamp: Date.now()
            } as AgentEvent);
          },
          onFinalAnswer: (answer) => {
            socket.emit('agent_event', {
              type: 'final_answer',
              data: { answer },
              timestamp: Date.now()
            } as AgentEvent);
          },
          onError: (error) => {
            socket.emit('agent_event', {
              type: 'error',
              data: { error },
              timestamp: Date.now()
            } as AgentEvent);
          },
          onDone: () => {
            socket.emit('agent_event', {
              type: 'done',
              data: null,
              timestamp: Date.now()
            } as AgentEvent);
          }
        };

        try {
          await chatWithAgent(data, callbacks);
        } catch (e: any) {
          callbacks.onError?.(e.message);
        }
      });

      socket.on('disconnect', () => {
        console.log('Client disconnected:', socket.id);
      });
    });
  }
}

export const webService = new WebService();
```

## 7. Web 端接入（Vue2）

```javascript
// web/src/services/websocket.js

import io from 'socket.io-client';

class AgentWebSocket {
  constructor() {
    this.socket = null;
    this.callbacks = {};
  }

  connect(url = 'http://localhost:40000') {
    this.socket = io(url);

    this.socket.on('agent_event', (event) => {
      const { type, data } = event;
      switch (type) {
        case 'thought':
          this.callbacks.onThought?.(data.thought, data.stepIndex);
          break;
        case 'action':
          this.callbacks.onAction?.(data.toolName, data.actionInput, data.stepIndex);
          break;
        case 'observation':
          this.callbacks.onObservation?.(data.result, data.isError, data.stepIndex);
          break;
        case 'final_answer':
          this.callbacks.onFinalAnswer?.(data.answer);
          break;
        case 'error':
          this.callbacks.onError?.(data.error);
          break;
        case 'done':
          this.callbacks.onDone?.();
          break;
      }
    });

    return this;
  }

  sendMessage(message, context = {}) {
    this.socket.emit('chat', { message, context });
  }

  onThought(callback) { this.callbacks.onThought = callback; return this; }
  onAction(callback) { this.callbacks.onAction = callback; return this; }
  onObservation(callback) { this.callbacks.onObservation = callback; return this; }
  onFinalAnswer(callback) { this.callbacks.onFinalAnswer = callback; return this; }
  onError(callback) { this.callbacks.onError = callback; return this; }
  onDone(callback) { this.callbacks.onDone = callback; return this; }

  disconnect() {
    this.socket?.disconnect();
  }
}

export default AgentWebSocket;
```

```vue
<template>
  <div>
    <div v-for="(t, i) in thoughts" :key="'t'+i">💭 {{ t }}</div>

    this.ws?.disconnect();
  }
```

## 8. CLI 端接入（Ink）

```typescript
// src/cli/cli.service.ts

import React, { useState, useEffect } from 'react';
import { render, Text, Box } from 'ink';
import { chatWithAgent } from '../api/ai.api';

interface Step {
  thought?: string;
  action?: { name: string; input: any };
  observation?: any;
}

const CLIStream: React.FC<{ message: string }> = ({ message }) => {
  const [steps, setSteps] = useState<Step[]>([]);
  const [finalAnswer, setFinalAnswer] = useState<string>('');

  useEffect(() => {
    chatWithAgent(
      { message, context: { projectPath: process.cwd() } },
      {
        onThought: (thought, stepIndex) => {
          setSteps(prev => {
            const newSteps = [...prev];
            newSteps[stepIndex - 1] = { ...newSteps[stepIndex - 1], thought };
            return newSteps;
          });
        },
        onAction: (toolName, actionInput, stepIndex) => {
          setSteps(prev => {
            const newSteps = [...prev];
            newSteps[stepIndex - 1] = { ...newSteps[stepIndex - 1], action: { name: toolName, input: actionInput } };
            return newSteps;
          });
        },
        onObservation: (result, isError, stepIndex) => {
          setSteps(prev => {
            const newSteps = [...prev];
            newSteps[stepIndex - 1] = { ...newSteps[stepIndex - 1], observation: result };
            return newSteps;
          });
        },
        onFinalAnswer: (answer) => setFinalAnswer(answer),
      }
    );
  }, [message]);

  return (
    <Box flexDirection="column">
      {steps.map((step, i) => (
        <Box key={i} flexDirection="column" marginY={1}>
          <Text color="cyan">💭 Step {i + 1}: {step.thought}</Text>
          {step.action && (
            <Text color="yellow">⚡ Action: {step.action.name}({JSON.stringify(step.action.input)})</Text>
          )}
          {step.observation && (
            <Text color={step.observation.error ? 'red' : 'green'}>
              📤 Result: {JSON.stringify(step.observation)}
            </Text>
          )}
        </Box>
      ))}
      {finalAnswer && <Text bold>✅ Final: {finalAnswer}</Text>}
    </Box>
  );
});

export function runCLI(message: string) {
  render(<CLIStream message={message} />);
}
```

## 9. 架构图

```
┌─────────────────────────────────────────────────────────────────┐
│                        统一 API 层                               │
│                                                                 │
│  chatWithAgent(req, callbacks)                                 │
│         │                                                       │
│         │  callbacks 是可选的                                   │
│         │  - 有 callbacks → 实时触发（CLI 模式）                 │
│         │  - 无 callbacks → 直接返回结果（无流式）               │
│         │                                                       │
│         ▼                                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    ReAct Agent                           │   │
│  │                                                          │   │
│  │  while (迭代) {                                          │   │
│  │    ai.chat() → thought/action                           │   │
│  │    callbacks.onThought?.()  ←── 实时触发                 │   │
│  │    tool.execute() → observation                         │   │
│  │    callbacks.onAction?.()                               │   │
│  │    callbacks.onObservation?.()                          │   │
│  │  }                                                       │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
          │                        │
          │ callbacks              │ callbacks → WebSocket
          ▼                        ▼
┌─────────────────────┐    ┌─────────────────────┐
│       CLI           │    │        Web          │
│   (Ink/React)       │    │   (Vue2 + Socket.io) │
│                     │    │                     │
│  直接打印 Thought   │    │  WebSocket 接收     │
│  直接打印 Action    │    │  渲染到浏览器        │
│  直接打印 Result    │    │                     │
└─────────────────────┘    └─────────────────────┘
```
