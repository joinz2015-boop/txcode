# TXCode 项目总览

## 1. 项目概述

- **项目名称**: `txcode` (CLI) / `txcode-web` (Web)
- **技术栈**: TypeScript + React + Ink (CLI) + Express + Vue2 + ElementUI (Web)
- **核心功能**: AI Coding Assistant，支持 CLI 和 Web 两种交互模式，采用 ReAct 模式与 AI 交互
- **目标用户**: 开发者2

## 5. 启动模式

```bash
txcode                    # CLI 交互模式
txcode "帮我写一个函数"    # CLI 单次对话
txcode web                # 启动 Web 服务 + 配置页面 (端口 40000)
```

---

## 6. 核心模块职责

| 模块 | 职责 |
|------|------|
| `db` | SQLite 数据库统一管理 |
| `memory` | 短期记忆（会话）+ 长期记忆（知识库） |
| `ai` | AI 服务、ReAct 模式、Tool 调用 |
| `session` | 会话管理 |
| `context` | 文件/项目上下文 |
| `skill` | Skill 加载（解析 .md 文件） |
| `tools` | 内置工具（文件读写、命令执行等） |
| `config` | AI 服务商/模型配置 |

---

## 7. 配置文件

```json
{
  "ai": {
    "maxToolIterations": 10,
    "maxSessionCompression": 5
  },
  "web": {
    "port": 40000
  }
}
```

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `ai.maxToolIterations` | 工具执行最大轮数 | 10 |
| `ai.maxSessionCompression` | 会话压缩保留消息数 | 5 |
| `web.port` | Web 服务端口 | 40000 |
