# 文件操作

txCode 内置了文件浏览器功能，可以在 Web 界面中浏览、编辑和管理文件。

## 进入文件页面

1. 启动 txCode Web 服务：

```bash
txcode web
```

2. 浏览器打开 `http://localhost:40000`
3. 点击左侧导航的 **Explorer** 进入文件浏览器

## 界面说明

### 文件树

左侧显示文件目录树，支持：

- 展开/折叠文件夹
- 显示文件图标（根据文件类型显示不同图标）
- 显示文件修改状态

### 编辑器

右侧显示文件编辑器，支持：

- 代码编辑器
- 语法高亮（支持 JS/TS/HTML/CSS/Vue/Python 等）
- 行号显示
- 快捷键支持

### 工具栏

文件树顶部工具栏：

| 按钮 | 功能 |
|------|------|
| ↑ | 上级目录 |
| ⟳ | 刷新 |
| 🏠 | 我的电脑（根目录） |

## 基本操作

### 浏览文件

1. 点击文件夹左侧箭头展开目录
2. 点击文件夹名称进入该目录
3. 双击文件打开编辑

### 打开多个文件

- 可以同时打开多个文件
- 点击标签页切换文件
- 点击标签页的 `×` 关闭文件
- 标签页显示文件名称和修改状态

### 编辑文件

1. 双击左侧文件打开编辑
2. 在编辑器中修改内容
3. 点击底部 **保存** 按钮或按 `Ctrl+S` 保存
4. 修改后文件名旁边显示 **已修改** 提示

### 创建文件

1. 右键点击目标文件夹
2. 选择 **新建文件**
3. 输入文件名
4. 按 Enter 确认

### 创建文件夹

1. 右键点击目标位置
2. 选择 **新建文件夹**
3. 输入文件夹名称
4. 按 Enter 确认

### 重命名

1. 右键点击文件或文件夹
2. 选择 **重命名**
3. 输入新名称
4. 按 Enter 确认

### 删除

1. 右键点击文件或文件夹
2. 选择 **删除**
3. 在确认对话框中点击 **删除**

### 复制路径

1. 右键点击文件或文件夹
2. 选择 **复制路径**
3. 路径已复制到剪贴板

## 支持的文件类型

| 扩展名 | 文件类型 | 图标 |
|--------|----------|------|
| .js | JavaScript | 黄色 JS 图标 |
| .ts | TypeScript | 蓝色 TS 图标 |
| .vue | Vue | 绿色 Vue 图标 |
| .py | Python | 蓝色 Python 图标 |
| .html | HTML | 橙色 HTML 图标 |
| .css | CSS | 蓝色 CSS 图标 |
| .json | JSON | 黄色文件图标 |
| .md | Markdown | 灰色文档图标 |
| .yaml/.yml | YAML | 蓝色 YAML 图标 |
| .sh | Shell | 绿色终端图标 |
| .go | Go | 青色图标 |
| .rs | Rust | 橙色图标 |

二进制文件（如图片、可执行文件）无法预览内容，会提示"二进制文件无法预览"。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存当前文件 |
| `Ctrl+W` | 关闭当前标签（需自行添加） |

## 注意事项

1. **权限**：文件操作权限与运行 txCode 的用户权限相同
2. **路径分隔符**：Windows 系统使用 `\`，Unix 系统使用 `/`
3. **编码**：默认使用 UTF-8 编码
4. **大文件**：大文件可能会有加载延迟
5. **自动保存**：不会自动保存，修改后需手动保存
