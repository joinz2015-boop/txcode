txcode操作文档 AI 日志

AI 日志

txCode 会记录所有 AI 模型调用日志，方便查看 token 消耗、耗时和费用。

进入日志页面

  1. 启动 txCode Web 服务：
txcode web
  1. 浏览器打开 http://localhost:40000
  2. 点击顶部导航的 AI 日志 进入日志查看页面

日志字段说明

字段 说明
ID 日志唯一标识
时间 请求发送的时间
类型 工具调用 / 普通对话
模型 使用的 AI 模型名称
耗时 请求处理时间（毫秒/秒）
输入Token 发送给模型的 token 数量
输出Token 模型返回的 token 数量
费用 本次调用的费用（美元）
会话ID 所属会话的标识

筛选条件

调用类型

  • 全部：显示所有日志
  • 工具调用：仅显示调用工具的请求
  • 普通对话：仅显示普通对话请求

会话

选择特定会话，查看该会话的所有 AI 调用记录。

时间范围

设置开始时间和结束时间，筛选指定时间段的日志。

日志统计

日志页面会显示：

  • 总记录数：当前筛选条件下的日志条数
  • 分页信息：支持翻页查看更多记录

Token 与费用

Token 说明

  • 输入 Token (input_tokens)：你的问题、上下文和提示词消耗的 token
  • 输出 Token (output_tokens)：AI 返回内容消耗的 token
  • 总 Token：输入 + 输出

费用计算

费用按模型定价计算，不同模型价格不同：

模型 输入价格 输出价格
GPT-4o $2.5/1M $10/1M
GPT-4o-mini $0.15/1M $0.60/1M
DeepSeek-chat $0.27/1M $1.10/1M

实际费用 = (输入Token × 输入单价 + 输出Token × 输出单价) / 1,000,000

日志管理

自动清理

系统自动管理日志容量：

  • 最大保存：5000 条日志
  • 清理阈值：超过 5000 条时，保留最近 1000 条

查看历史

使用时间筛选功能，可以查看任意时间段的调用记录。

常见问题

Q: 为什么费用显示为 0？
A: 某些模型的费用计算可能未完整，或者 token 数为 0。

Q: 如何查看某个会话的消耗？
A: 在会话筛选中选择该会话，即可查看该会话的所有 AI 调用记录。

Q: 日志保留多长时间？
A: 日志保留最近 5000 条，超过后自动清理旧记录。
