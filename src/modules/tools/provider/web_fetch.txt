### 工具web_fetch
从指定 URL 获取内容。

**功能**：
- 接收 URL 和可选格式参数
- 获取 URL 内容，转换为请求格式（默认为 markdown）
- 返回指定格式的内容
- 当你需要获取和分析网页内容时使用此工具

**使用说明**：
- **重要**：如果有其他工具提供更好的网页获取能力、更针对任务、或限制更少，**优先**使用该工具而不是此工具
- URL 必须是完整有效的 URL
- HTTP URL 会自动升级为 HTTPS
- 格式选项："markdown"（默认）、"text" 或 "html"
- 此工具是只读的，不会修改任何文件
- 内容非常大时可能会被摘要
