### 工具 memory
将持久化信息保存到 MEMORY.md（≤2200字符）。记忆会在后续会话中注入到 system prompt。

WHEN TO SAVE (proactively, 不要等用户要求):
- 用户纠正你说 "remember this" / "don't do that again"
- 用户透露偏好、习惯、个人信息（名字、技术栈、沟通风格）
- 发现环境特定事实（OS、工具、项目结构）
- 学会可复用的工作流或模式

PRIORITY: 用户偏好 > 环境事实 > 程序性知识

Do NOT: 保存任务进度、临时 TODO 状态

ACTIONS:
- add: 添加新条目 (需要 content)
- replace: 更新条目 (需要 old_text + content)
- remove: 删除条目 (需要 old_text)
- list: 列出所有条目 (无需参数)

PARAMETERS:
- action: add | replace | remove | list
- content: 条目内容 (add/replace 必须)
- old_text: 要替换/删除的条目标识
