使用 Exa AI 搜索网络——执行实时网络搜索，可以抓取特定 URL 的内容。

**功能**：
- 提供最新信息用于时事和近期数据
- 支持可配置的结果数量，返回最相关网站的内容
- 用于访问知识截止日期之外的信息
- 在单个 API 调用内执行搜索

**使用说明**：
- 支持实时爬取模式：'fallback'（缓存不可用时备用）或 'preferred'（优先实时爬取）
- 搜索类型：'auto'（平衡）、'fast'（快速结果）、'deep'（全面搜索）
- 可配置上下文长度以优化 LLM 集成
- 支持域名过滤和高级搜索选项
