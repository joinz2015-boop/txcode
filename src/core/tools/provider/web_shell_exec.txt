### 工具web_shell_exec
在当前已连接的远程主机上执行 shell 命令。命令和输出会同步显示在 WebShell 终端中。

**功能**：
- 在远程 SSH 主机上执行 shell 命令
- 适用于系统运维、日志分析、性能诊断等场景
- 命令输出实时推送到 WebShell 终端
- 支持设置超时时间

**使用说明**：
- 此工具依赖当前 WebSocket 会话已关联的 SSH 连接，不需要指定主机
- 如需状态连续性（如 cd 后执行），请拼接为 `cd /path && command`
- 默认超时 30 秒，长时间运行的命令请指定更长的超时
- 输出最大 50KB / 2000 行，超出会被截断
- 谨慎执行可能造成系统影响的命令（如 rm、shutdown 等）
